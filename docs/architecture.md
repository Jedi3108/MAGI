# MAGI Architecture

MAGI is organized around a separation of reasoning, memory, tools, execution, and interface.

Decision Dossier
Council

The council consists of four permanent members:

MELCHIOR
BALTHASAR
CASPER
ARTABAN

Each member has a stable identity but can run on different underlying models.

Protocol

The protocol defines how the council deliberates:

Independent analysis
Cross-examination
Directed responses
Satisfaction check
Reflection
Final vote
Chair summary
Chair

The Chair does not vote. It summarizes the deliberation and produces the official decision dossier.

Memory

The memory layer stores decisions, traces, precedents, and later reliability information.

Workers

Workers execute approved actions. Council members do not directly act on the world.

Principle

The council reasons. The world acts.
