# v0.1.0-alpha — The First Council

## Status

`v0.1.0-alpha` is the first functional MAGI protocol release.

It establishes a complete mock-mode deliberation loop with four permanent council members and a non-voting Chair.

This release is not intended to demonstrate intelligence quality. It demonstrates protocol correctness, traceability, and architectural direction.

## Council Members

MAGI currently consists of four voting council members:

- **MELCHIOR** — The Scientist  
  Truth, evidence, technical correctness, epistemic discipline.

- **BALTHASAR** — The Mother  
  Care, safety, wellbeing, sustainability, protection.

- **CASPER** — The Woman  
  Individuality, intuition, aesthetics, desire, lived meaning.

- **ARTABAN** — The Man  
  Duty, execution, resolve, responsibility, consequence.

## Non-Voting Chair

The Chair does not vote.

The Chair summarizes the actual council deliberation into a decision dossier.

The Chair is responsible for clarity, traceability, and faithful synthesis.

## Current Protocol

The current deliberation protocol contains five stages:

1. **Independent Analysis**  
   Each council member independently evaluates the proposition.

2. **Cross-Examination**  
   Council members may ask directed questions to other members.

3. **Satisfaction Evaluation**  
   The original asker evaluates whether the answer resolved their concern.

4. **Reflection**  
   Each council member reflects on the exchange and may revise vote or confidence.

5. **Chair Dossier**  
   The non-voting Chair produces a final decision dossier.

## Decision Dossier

The Chair dossier includes:

- Final decision
- Vote split
- Majority reasoning
- Minority reasoning
- Key risks
- Outstanding uncertainties
- Required conditions
- Recommended next action

## Current Backend

This release supports:

- Mock backend for deterministic protocol testing
- Ollama backend for local model execution

The mock backend is intentionally robotic and repetitive. It exists to test the protocol flow without depending on model quality.
