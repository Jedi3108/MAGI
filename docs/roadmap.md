# MAGI Roadmap

## v0.0.1 — Genesis

Foundation repository, documentation structure, council identity, and project constitution.

## v0.1 — The First Council

Goal: create the first stable local deliberative council.

Planned capabilities:

- Four permanent council members
- Structured council outputs
- Independent analysis
- Cross-examination
- Reflection
- Final vote
- Chair summary
- Local mock mode
- Local Ollama backend

## v0.2 — Institutional Memory

- Decision logs
- Precedent search
- Minority reports
- Confidence history

## v0.3 — Tool Layer

- File access
- Python execution
- Web/search interface
- Git integration

## v0.4 — Executive Layer

- Worker agents
- Approved action execution
- Human confirmation gates

## v0.5 — Specialist Advisors

- Domain-specific consultants
- Research advisor
- Programming advisor
- Mathematics/statistics advisor

## v0.6 — NERV Interface

- Local web interface
- Council visualization
- Debate timeline
- Decision dossier display

## v1.0 — Daily Driver

A stable local assistant that the user can rely on for research, planning, coding, writing, and decision support.
